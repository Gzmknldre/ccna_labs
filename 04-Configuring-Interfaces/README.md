# Configuring Interfaces

## Objective
Configure switch interfaces and verify their status.

## Commands Used

- `show ip interface brief`
- `show interfaces status`
- `interface range`
- `speed`
- `duplex`

## What I Learned

- `show ip interface brief` displays interface IP addresses and their status.
- `show interfaces status` provides a summary of switch port status, VLANs, speed, and duplex settings.
- `interface range` allows multiple interfaces to be configured at the same time.
- Speed and duplex can be configured manually instead of using auto-negotiation.
- Interface configuration can be applied to multiple ports simultaneously using `interface range`.
