## Commands Used

- `ip address <ip-address> <subnet-mask>`
- `description <text>`
- `no shutdown`
- `show ip interface brief`

## What I Learned

- Each router interface must have its own IP address.
- Each interface should be configured with the correct subnet mask.
- The `description` command helps identify what a router interface is connected to.
- An interface remains administratively down until the `no shutdown` command is used.
- `show ip interface brief` can be used to verify interface status and IP addresses.
